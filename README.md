# Language Learning Dashboard

A comprehensive web application designed to help users learn new languages through an interactive dashboard, vocabulary management, and quiz functionality.


## Demo
![Alt Text](https://github.com/tanczacy-ziemniak/language-learning-dashboard/blob/main/demo.gif)


## Features

- **Dashboard**: Overview of learning progress and quick access to all features
- **Vocabulary Management**: Add, edit, and organize words and expressions
- **Notes**: Create and manage study notes
- **Quiz System**: Test your knowledge with various quiz types
- **Content Manager**: Organize all your learning materials in one place
- **Responsive Design**: Works on desktop and mobile devices

## Technologies Used

- React.js
- React Router
- CSS Modules
- Local Storage for data persistence

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/tanczacy-ziemniak/language-learning-dashboard.git
   ```

2. Navigate to the project directory:
   ```bash
   cd language-learning-dashboard
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm start
   ```

5. Open your browser and visit:
   ```
   http://localhost:3000
   ```

## Usage

### Dashboard
The dashboard provides an overview of your learning progress and quick access to all features.

### Words and Expressions
Add new vocabulary items with translations, example sentences, and categories for organized learning.

### Notes
Create and manage study notes for grammar rules, cultural insights, or any other language learning content.

### Quiz
Test your knowledge with different quiz types:
- Multiple choice questions 


### Content Manager
Organize all your learning materials by categories, difficulty levels, or custom tags.
